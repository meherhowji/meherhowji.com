## Technologies

- Language: JS, HTML & CSS
- Base Library: React
- Framework: NextJS
- Content: Markdown
- Content Database: Obsidian
- Renderer: MDX
- Styling: Bulma
- Animations: React Spring
- Theme: Designed by me, Meher Howji
- Deployment: Vercel
